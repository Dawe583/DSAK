Časté dotazy

Jasné technické a provozní parametry toho, jak stavíme, nasazujeme a zabezpečujeme vaši architekturu. Bez omáčky.

✉ ahoj@branzly.cz
Jsou naše firemní data v bezpečí před veřejnými AI modely?+

Ano. Používáme API bez retence dat a nasazujeme výhradně ve vašem privátním cloudu. Vaše data jsou oplocená a nikdy netrénují veřejné modely.

Integruje se to s naším starším softwarem?+

Ano. Pro každý systém s API stavíme vlastní middleware. U uzavřeného staršího softwaru nasazujeme bezpečnou RPA vrstvu, která mezeru překlene.

Jak rychlé je nasazení?+

Audit trvá 7 dní. Interní automatizace nasazujeme za 3 až 4 týdny. Agenti pro komunikaci se zákazníky startují za 6 až 8 týdnů včetně testování.

Nahradí to naše zaměstnance?+

Ne. Naše systémy samy vyřeší zhruba 80 % opakujících se úkolů první úrovně. Složité případy se okamžitě předávají vašim lidem.

Komu patří kód a duševní vlastnictví?+

Po dokončení vlastníte 100 % IP. Nabízíme průběžné SLA na monitoring stability API a aktualizace podkladových jazykových modelů.
